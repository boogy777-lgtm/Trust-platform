# Contribute

## Public Contribution Paths

- file bugs and improvement requests in GitHub Issues
- improve docs and examples
- propose or review workflow fixes
- contribute code with tests

Project home:

- <https://github.com/boogy777-lgtm/Trust-platform>

## Good First Contribution Areas

- docs wording, screenshots, and workflow gaps
- example README fixes
- search/discoverability improvements
- CLI/reference consistency fixes
