# Runbooks

Site-specific runbooks should include:

- the HMI URL
- local credentials or access path
- escalation contacts
- start-of-shift checks
- alarm procedures
- shift handover notes

Adapt the template after the HMI URL, contacts, and escalation rules are known.
Pair it with [Operator Daily Checks](../operate/operator-daily-checks.md) and
[Operator Alarm Handbook](../operate/operator-alarm-handbook.md).
