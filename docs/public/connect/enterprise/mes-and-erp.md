# MES And ERP

## Boundary Rule

truST is strongest at:

- control/runtime execution
- plant-facing protocols
- browser and operator UIs
- CLI/agent automation

MES/ERP integration usually happens through one of these boundaries:

- protocol bridge already accepted by the site
- CLI or agent automation
- historian/report export path

## Safest First Test

1. prove the runtime state locally
2. define the small business event or payload you need
3. automate one read or write path
4. capture the full before/after result
