# Runtime Cloud

Use this manual when:

- the cloud/federation wiring already exists
- you need the operator/developer manual for daily use
- you are troubleshooting runtime-cloud actions, status, or policy in operation

The manual is for daily use of an existing runtime-cloud setup; the quickstart
is for first proof. If you are still deciding whether federation is the right
architecture, start with [Runtime Cloud Federation](../connect/runtime-to-runtime/runtime-cloud-federation.md).

Success means an operator or maintainer can explain the active topology, run
the expected runtime-cloud action, and identify whether a failure is topology,
policy, credentials, or runtime state.

Use [Observability](observability.md) next when the daily-use question turns
into evidence collection over time.
